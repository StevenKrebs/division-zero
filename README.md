# division-zero
Covers the development of the website available @ division-zero.org

[![Build Status](https://travis-ci.org/StevenKrebs/division-zero.svg?branch=master)](https://travis-ci.org/StevenKrebs/division-zero)

### Dependencies
 - run `npm install`

### Build steps

##### Dev
 - run `gulp serve --dev`

##### Deploy
 - run `gulp build`
 - deploy out of `./dist`
